# Welcome to APP AdmSys Backend
### Deploy to Heroku [BackAdmSys](https://backadmsys.herokuapp.com/).

 The Backend of  App **Admsys** is created with love & uses technologies such as:
 - NodeJS.
 - Express.
 - MongoDB.
 - Mongoose.
 - JsonWebToken.
 - BcryptJS.
 - Dotenv.
 - Body Parser.

Use Postman for request:
